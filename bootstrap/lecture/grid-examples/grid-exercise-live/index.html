Bootstrap Grid Layout Exercises for COSC3060

Exercise 1: Simple Two-Column Layout

Goal: Create two columns that take up 50% of the width on extra-large screens (1200px and up) and 100% on smaller screens (stacked vertically).

Hint: Use the col-xl-6 class for each column.

The target layout:

50% desktop, 100% mobile
50% desktop, 100% mobile

Your current layout:

50% desktop, 100% mobile
50% desktop, 100% mobile
Show Solution 
<div class="row">
  <div class="col-xl-6">50% desktop, 100% mobile</div>
  <div class="col-xl-6">50% desktop, 100% mobile</div>
</div>

Explanation:
- The .row class creates a flex container to hold the columns.
- .col-xl-6 divides the row into 12 equal parts and assigns 6 parts to each column (50% each) on extra-large screens.
- On smaller screens, columns automatically stack.

Exercise 10: Complex Nested and Responsive Layout

Goal: Build a complex layout combining several Bootstrap grid features. Create a parent row with two columns. In the left column, create a nested grid with two rows. The first nested row should have two columns (one using col-8 and the other col-4). The second nested row should have three equal columns. The right column should display a single centered column using offset classes.

Hint: Work step-by-step. First, create the main two-column row. Then, focus on the nested grid in the left column and finally apply offsets in the right column.

The target layout:

Nested Row 1, Col 1
Nested Row 1, Col 2
Nested Row 2, Col 1
Nested Row 2, Col 2
Nested Row 2, Col 3
Right Column

Your current layout:

Left Column
Right Column
Show Solution 
<div class="row">
  <div class="col-lg-8">
    <div class="row mb-3">
      <div class="col-8">Nested Row 1, Col 1</div>
      <div class="col-4">Nested Row 1, Col 2</div>
    </div>
    <div class="row">
      <div class="col">Nested Row 2, Col 1</div>
      <div class="col">Nested Row 2, Col 2</div>
      <div class="col">Nested Row 2, Col 3</div>
    </div>
  </div>
  <div class="col-lg-4">
    <div class="row">
      <div class="col-8 offset-2">Right Column</div>
    </div>
  </div>
</div>

Explanation:
- The parent .row splits the container into two columns.
- In the left column (.col-lg-8), two nested rows are created:
  • The first nested row uses .col-8 and .col-4 for an 8/4 split.
  • The second nested row uses .col for equal-width columns.
- The right column (.col-lg-4) centers its content using offset classes (offset-2).
